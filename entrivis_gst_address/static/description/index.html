Fetch Address From GST Number

This module fetches address and PAN number from GST number of a partner.

  • First install the module and go to company configuration, enter appyflow secret key.

  • To get your appyflow secret key Click Here

  • Register with required details. After registring go to the account and get your Appyflow secret key. Purchase credits to get started.

  • Create a contact or a customer or a vendor, enter name and GST number. Click on Fetch Address button.Verify the address and save the record.

  • The registered address and PAN number will be auto populated.
